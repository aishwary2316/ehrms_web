## How to Login

1. Go to the login page
2. Enter your **EIN** (e.g., `sp001`, `ins001`, `asi0001`) as username
3. Enter your **first name in lowercase** (e.g., `joykumar`, `hemanta`) as password
4. Click Login

### Examples:
- **SP Level**: Username: `sp001` | Password: `joykumar`
- **ASP Level**: Username: `asp001` | Password: `ratan`
- **Inspector**: Username: `ins001` | Password: `hemanta`
- **Constable**: Username: `con0001` | Password: (their first name)

## Password Policy

- All passwords are currently set to first names for easy access
- **IMPORTANT**: Users should change their passwords upon first login
- Administrators can reset passwords through the user management interface

## Access Levels by Role

1. **SP**: Full system access, can manage all users and data
2. **ASP**: Administrative access, can manage divisional operations
3. **SDPO**: Divisional access, can manage sub-divisions
4. **Inspector**: Station-level access, can manage station operations
5. **Personnel**: Basic access, can view own data and apply for leaves
